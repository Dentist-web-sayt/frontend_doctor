# DentaLux — Zamonaviy Stomatologiya Klinikasi Sayti

## Texnologiyalar
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (ixtiyoriy)

## O'rnatish

```bash
npm install
npm run dev
```

## Sahifalar
- `/` — Bosh sahifa (Hero, Xizmatlar, Shifokorlar, Sharhlar)
- `/services` — Barcha xizmatlar
- `/doctors` — Shifokorlar jamoasi
- `/booking` — Navbat olish (3-bosqichli forma)
- `/contact` — Aloqa va xarita

## Xususiyatlar
- ✅ Dark / Light mode
- ✅ To'liq responsive (mobile, tablet, desktop)
- ✅ Scroll reveal animatsiyalar (har tarafdan)
- ✅ Premium minimalist dizayn
- ✅ DM Serif Display + Sora shriftlari
- ✅ 3-bosqichli booking forma
